Stimulus Flatpickr Playground

Basic

Date time

Human-friendly Dates

        <input
          type="text"
          placeholder="Select Date.."
          data-controller="flatpickr"
          data-flatpickr-date-format="Y-m-d"
          data-flatpickr-alt-format="F j, Y"
          data-flatpickr-alt-input="true"
        />

minDate and maxDate

        <input 
          type="text" 
          placeholder="Select Date.." 
          data-controller="flatpickr" 
          data-flatpickr-min-date="2020-01" 
        />

Flatpickr + external elements

if you need to wrap custom elements inside the controller you can use two solutions:

1) Wrap: true option
        <div data-controller="flatpickr" data-flatpickr-wrap="true">
          <input type="text" placeholder="Select Date.." data-input />
          <button title="toggle" data-toggle>open</button>
          <button title="clear" data-clear>clear</button>
        </div>
2) Instance target
        <div data-controller="flatpickr">
          <input type="text" placeholder="Select Date.." data-target="flatpickr.instance" />
        </div>
